Let's Learn about Different Programming Languages!

What is Programming

Programming is how apps, websites, and computer software are made (just like this one). There are many types of coding, but some of the most popular ones are:

Python

Python is a high - level language that is object oriented. It is typically used for back-end (behind the scenes) web development, data analysis, artificial intelligence, and scientific computing

HTML and CSS (this webpage)

HTML is the standard markup language for web pages. It creates what you see when you arrive at a website. CSS describes how HTML elements are displayed (color, size, font, etc.)

Java

Java is considered the dominant language in coding. It is useful for web and app development

Javascript

Javascript is the programming language for the web. This language makes it easy to move code from one operating system to another without modifications.

Swift

Swift was written by Apple Inc. to be used mainly with Apple products.
